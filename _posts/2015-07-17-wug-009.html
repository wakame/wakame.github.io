OpenVNet 0.7 presentation and live demostration

In Japanese by Andreas Kieckens.

Recently OpenVNet has released version 0.7 which is its first stable release. OpenVNet can be thought of as a network hypervisor. It allows fully virtualized networks using OpenFlow. During this presentation, Andreas will briefly explain the concept of virtual networking. Afterwards he will show a live demo where OpenVNet is used to join two IaaS cloud systems (Wakame-vdc and Amazon EC2) in a single L2 network.

OpenVNet source code reading

In Japanese by Andreas Kieckens.

During this presentation we will delve into the OpenVNet source code. While looking at the code, we will give detailed information on how OpenVNet works. Parts of this will include class hierarchy, WebAPI validation, event handling, flow creation/deletion, etc.
